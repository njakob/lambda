
# lambda [![Build Status][build-status-image]][travis] [![ESLint Config][eslint-config-image]][eslint-config]

> Utils to deploy AWS Lambda functions written in Javascript.

:rotating_light: WIP

## Features

* CLI
* Public API
* Flowtype definition

## Installation

With NPM:

```
$ npm install @njakob/lambda
```

With Yarn:

```
$ yarn add @njakob/lambda
```

## Usage

```sh
$ lambda show
```

```sh
$ lambda pack
```
